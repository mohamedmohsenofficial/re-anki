## 💡 Why This Matters: The Future of Learning?

We're not just making flashcards prettier—we're revolutionizing how the brain interacts with digital learning tools. This project represents a shift from simple information delivery to experience design, combining neuroscience, design psychology, and mobile-first accessibility.

## 🧠 Backed by Cognitive Science

- **Dual Coding Theory (Paivio, 1971):** Combining visuals + text = stronger memory
- **Aesthetic-Usability Effect:** Attractive cards feel easier to use, keeping you engaged longer
- **Color Psychology:** Research shows color boosts memory retention by **82%** (Loyola University)
- **Picture Superiority Effect:** Visual animations improve encoding & recall

📚 References:

- Mayer, R. (2014). _Cognitive Theory of Multimedia Learning_
- Sweller, J. (1994). _Cognitive Load Theory_
- Elliot & Maier (2007). _Color and Psychological Functioning_
- Norman, D. (2004). _Emotional Design_
